PREVIEW IMAGE PLACEHOLDER

For workshop publication, you need to add a preview image here named "Preview.edds".

Requirements:
- Image should be 512x512 pixels
- Format must be .edds (Enfusion texture format)
- File must be named exactly "Preview.edds"

You can create this image using:
1. Arma Reforger Tools Texture Processor
2. Any image editing software that can export to .edds format
3. Converting from a .png or .jpg using the Workbench texture conversion tools

This image will be displayed on the Steam Workshop page for your mod.
